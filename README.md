# DropComp 2.0

After Effects CEP panel: a personal library of reusable comps with thumbnails.
Source of truth for the extension installed at
`~/Library/Application Support/Adobe/CEP/extensions/DropComp`.

- `./dev-link.command` - symlink this repo as the live extension (development)
- `./install.command` - copy files into the extensions folder (stable install)
- `npm test` - run unit tests (node >= 18)

Full design: `docs/superpowers/specs/2026-06-11-dropcomp-2-refinement-design.md`
